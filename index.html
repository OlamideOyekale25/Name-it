How to Play

  • Share the game code with friends so they can join
  • One player starts the round to pick a random letter
  • All players have 45 seconds to write names for People, Things, Animals, and Places starting with that letter
  • Submit your answers before time runs out or wait for the timer
  • After everyone submits, review and mark correct answers together
  • Each correct answer = 5 points
